# Solaris!

Solaris is a super customizable XP bot for Discord with all sorts of neat features!

## Transferring data from the original Polaris

**NOTE:** The import feature is heavenly limited for security reasons so feel free to join my [Discord server](https://discord.gg/B82EbWxbf2) for help to import your previous data.

1. On the [original Polaris dashboard](https://gdcolon.com/polaris), go to the Data tab of your server settings and press **Download all data**. This will download a .json file.
2. On the [Solaris dashboard](https://solaris.iku.pet/) (or your own hosted dashboard), go to the Data tab of your server settings, scroll down, and scroll down to the import settings section.
3. Upload the .json file and press import. (If it gives you an error read the note above.)
4. All data from Polaris should be transferred!
